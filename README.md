# ali-sum

A web app that scans your AliExpress account's order history and displays expense analytics.

## Overview

ali-sum connects to your AliExpress account, reads your order history, and provides a dashboard with spending analytics — totals, breakdowns by category/time period, and visual charts to help you understand your purchasing habits.

Inspired by [ali-t](https://github.com/RonenMars/ali-t).

## Planned Features

- AliExpress order history scanning
- Expense totals and summaries
- Spending breakdowns (by category, time period, seller)
- Visual charts and analytics dashboard
- Export/download reports
